|      |                               | 命令              |
| ---- | ----------------------------- | --------------- |
| 删除文件 | 删除 filename 之外的文件             | rm !(filename)  |
|      | 删除 C 源代码以外的文件（可能需要设置bash zsh） | `rm -v !(*c)`   |
| 删除目录 | 删除 directory                  | rm -r directory |
