### 文本域标签 textarea

**textarea文本域标签:**

- 场景:在网页中提供可输入多行文本的表单控件
- 标签名: textarea
- 常见属性:
  - cols:规定了文本域内可见宽度
  - rows:规定了文本域内可见行数
- 注意点:
  - 右下角可以拖拽改变大小
  - **实际开发时针对于样式效果推荐用CSS设置, 不用属性, 要用标签**

<img src="https://illyber-images.oss-cn-chengdu.aliyuncs.com/202302040340657.png" alt="image-20230204034003583" style="zoom:80%;" />
```html
<textarea name="" id="" cols="5" rows="4"></textarea>
```

# 选择

### 11-单选属性值 radio
- name 属性名：用来分组，一组只能选一个；
- checked 属性：默认选中。

```html
性别: <input type="radio" name="gender" checked>男
	 <input type="radio" name="gender">女
```

### 11-多选属性值 checkbox
- name 属性名：用来分组，一组可以选多个；
- checked 属性：默认选中。

# 提交
### 13-input 标签的按钮属性

![image.png](https://illyber-images.oss-cn-chengdu.aliyuncs.com/20240408183545.png)
- type属性值:

| 标签名 | type属性值 |                   说明                   |
| :----: | :--------: | :--------------------------------------: |
| input  |   submit   |  提交按钮。点击之后提交数据给后端服务器  |
| input  |   reset    |     重置按钮。点击之后恢复表单默认值     |
| input  |   button   | 普通按钮。默认无功能，之后配合js添加功能 |
|        |   value    |            设置按钮显示的文字            |
```html
user:     <input type="text"><br>
password: <input type="password"><br><br>
    <input type="submit" value="免费注册">
    <input type="reset">
    <input type="button" value="普通按钮">
```
- 注意点:
  - 如果需要实现以上按钮功能，需要配合form标签使用
  - form使用方法:用form标签把表单标签一起包裹起来即可

### 14-自定义按钮标签button

![image.png](https://illyber-images.oss-cn-chengdu.aliyuncs.com/20240408183732.png)
```html
<button>我是按钮</button>
```

**button既可以作type的属性值, 也可以自己做标签, type做button的属性名; 双标签**

- 注意点:

  - button标签是双标签，更便于包裹其他内容:文字、图片等

# 绑定文字至标签
### 17-label标签

![image.png](https://illyber-images.oss-cn-chengdu.aliyuncs.com/20240408184424.png)

```html
性别:   <input type="radio" name="gender" id="man">
        <label for="man">男</label >
            
        <label><input type="radio" name="gender">女</label>
```
- 场景:常用于绑定内容与表单标签的关系

- 标签名: label
- 使用方法1:
  - 1.使用label标签把内容(如:文本)包裹起来
  - 2.在表单标签上添加id属性, **id可以是字符串**
  - 3.在label标签的for属性中设置对应的id属性值
- 使用方法2:
  - 1.直接使用label标签把内容(如:文本)和表单标签—起包裹起来
  - 2.需要把label标签的for属性删除即可
