Zsh 快捷键

即zsh line editor
与 gnu readline library 不一致的快捷键：

快捷键 zle readline
ctrl+u 删除一整行 删除光标到行首的内容

运行zsh初始设置

补全一个单词：alt+f. 设计得和gnu readline一致

zsh插件和主题

zsh-autosuggestions

A Guide to the Zsh Line Editor with Examples

  • ^ - Represent the CTRL key. For example: ^c for CTRL+c.
  • \e - Represent the ALT key. For example: \ec for ALT+c.

zsh 提示符

参考：布_鲁斯 - zsh 命令提示符 PROMPT, 木_穆 - 自定义 zsh 提示（配置PROMPT或PS1）

在 ~/.zshrc 里设置

例子：export PROMPT='%B%F{green}%n@%M%f%b:%B%F{blue}%/%f%b$ '

  • %F{color} 是配置颜色，{}中color是 256 色的颜色值，也可以使用black，red，green，yellow，blue，magenta，cyan和white等常用色。
  • %f 表示后面恢复默认颜色。
  • %B 粗体 。
  • %b 表示后面恢复默认子重。
