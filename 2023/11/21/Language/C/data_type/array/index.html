变长数组(VLA)

VLA 是 C99 新增的特性
int ar[n]

  • C90 不允许 n 为 const 类型
  • C99/C11 允许 n 为 const 类型
  • 变长数组不能初始化
  • 变长数组必须为自动存储类别，离开所在块后即被释放；也就是说不能用 static 和 extern 关键字
  • 变长数组创建完成后，长度不可改变；可变指的是创建时可以用变量指定数组的维度

数组与指针

  1. 数组名是指向数组元素的指针，因为要用数组名遍历数组。
  • ar 是 int *;
  • num[0] 是 int *, num 是 int (*)[10]
  1. 多维数组的低维可以看做高维的元素。
