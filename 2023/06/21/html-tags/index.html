html标签总结

发表于2023-06-21|更新于2023-06-21|languagehtml
|阅读量:

HTML入门教程（非常详细）

基本骨架

  • 单标签, 双标签(开始标签, 结束标签)
  • 属性名, 属性值

排版标签

标题标签

段落标签

paragraph

换行标签

break row

水平尺标签

horizontal rule

文本格式化标签

表2表示强调, 给程序员看

标签 说明 英文全称
b 加粗 bold
u 下划线 underlined
i 倾斜 italic
s 删除线 strikethrough
标签 说明 英文全称
strong 加粗
ins 下划线 inserted
em 倾斜 emphasized
del 删除线 deleted

媒体标签

图片

  • 标签属性
标签名 作用
src 图片路径
alt 替换文本: 图片不加载时显示的文本
title 悬停时显示的文本
width 宽度, 默认以像素为单位
height 高度, 默认以像素为单位

音频和视频

标签名 src controls autoplay loop muted
audio 文件路径 显示播放控件 自动播放(Chrome不行) 循环播放
video 文件路径 显示播放控件 自动播放(需配合muted) 循环播放 静音

超链接标签

属性名 属性值 作用
href 网页位置 空连接用#
target _self 默认值, 在当前标签页打开新网页
_blank 在新标签页打开新网页

列表标签

列表标签 列表术语 项标签
无序列表 ul(unordered list) li(list item)
有序列表 ol(ordered list) li(list item)
自定义列表 dl(definition list) dt(definition term) dd(definition description)
  • 注意: 修饰文字的标签放在li/dd里

表格标签

基本标签:

标签 含义
table 表格整体
tr 表格行(table row)
td 表格数据(table data cell)

table的属性:

属性名 作用 属性值
border 边框 边框粗细, 默认单位为像素px
width 表格整体宽度 数字, 默认单位为像素px
height 表格整体高度 数字, 默认单位为像素px

td的属性:

属性名 作用 属性值
width 表格单元宽度 数字, 默认单位为像素px
height 表格单元高度 数字, 默认单位为像素px

用td的属性合并表格:

  • td属性
  • 需要按左上保留原则, 手动删除多余的表格
属性名 属性值 作用
rowspan 合并的表格数 跨行合并, 将多行的单元格垂直合并
colspan 合并的表格数 跨列合并, 将多列的单元格垂直合并

描述表格的标签:

标签 含义 位置
caption 表格整体的标题 table内, tr外
th 表头(居中加粗) 替换td

结构标签:

标签 含义 作用
thead 表格头部 缩小高度, 不会加粗
tbody 表格主题 增加高度
tfoot 表格底部 缩小高度, 不会加粗

表单

场景:

  • 输入文本(单行输入框, 密码, 多行输入)
  • 选择选项(单选, 多选)
  • 选择文件
  • 下拉菜单
  • 按钮
  • label

HTML表单：form标签form放在最外面?
form：表单域标签，提交、重置等按钮对form里面的表单起作用

几种属性: value（给后端、文字）, name(分组）, id（label for）

输入单行文本和密码:

标签名 属性名 属性值 说明
input type text 文本框，用于输入单行文本
input type password 密码框, 用于输入密码
placeholder 提示文本 占位符。提示用户输入内容的文本

选择:

标签名 属性名 属性值 说明
input type radio 单选框，用于多选
input type checkbox 多选框，用于多选多
name 可以是字符串 分组。有相同name属性值的单选框为一组，一组中同时只能有一个被选中
checked 默认选中

上传文件:

标签名 属性名 属性值 说明
input type file 文件选择，用于之后上传文件
multiple 多选文件

按钮:

标签名 属性名 属性值 说明
form
input type submit 提交按钮，用于提交
input type reset 重置按钮，用于重置
input type button 普通按钮，默认无功能，之后配合js添加功能
value 设置按钮显示的文字
button type submit 提交按钮，用于提交
button type reset 重置按钮，用于重置
button type button 普通按钮，默认无功能，之后配合js添加功能

下拉菜单:

标签名 属性名 属性值 说明
select 下拉菜单的整体
option 下拉菜单的每一项
selected 下拉菜单的默认选中

文本域:

标签名 属性名 属性值 说明
textarea cols 数字 规定了文本域内可见宽度
rows 数字 规定了文本域内可见行数

label标签:

  • 场景:常用于绑定内容与表单标签的关系

  • 标签名: label

  • 使用方法①:

    • 1.使用label标签把内容(如:文本)包裹起来
    • 2.在表单标签上添加id属性, id可以是字符串
    • 3.在label标签的for属性中设置对应的id属性值

  • 使用方法②:

    • 1.直接使用label标签把内容(如:文本)和表单标签—起包裹起来
    • 2.需要把label标签的for属性删除即可
