字体大小

可直接复制配置文件。

  1. 编辑器字体大小：右下角齿轮->设置->搜索字体->Editor:Font Sizer 里写上18

  1. 系统界面缩放。在设置里搜索「缩放」

  2. 终端字体大小：在设置里搜索 Terminal › Integrated: Font Size

如何运行/停止？

  1. 编辑区->右键->Run Code
  2. 停止：输出区->右键->stop run code

技巧

  • !+tab or !+enter. 建立框架
  • Ctrl + d. 同时选中改变多个
  • 查看->自动换行. 可使较长的代码自动换行
  • ctrl+enter;ctrl+shift+enter分别是另外开一行到下一行和上一行
  • li*5直接生成5行li
  • 快速移到右边按end(fn+->)
  • 安装htmltagwrap. 自动更改两侧标签, alt+w, 在选中的文本的两端添加标签
  • 缩进. 用光标选中+tab.
  • 取消缩进. 用光标选中+shift+tab
  • 注释: Ctrl+/

折叠代码

  • 折叠所有代码：ctrl+k, ctrl_0
